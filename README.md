# codealpha_language-translation
A modern AI-powered Language Translation Tool built with HTML, CSS, and JavaScript featuring multi-language translation, language detection, text-to-speech, clipboard support, and a responsive user interface.
## ✨ Features

- 🌍 Translate between multiple languages
- 🔍 Automatic language detection
- 🔄 Swap source and target languages
- 🔊 Text-to-Speech (TTS)
- 📋 Copy translated text
- ⌨️ Keyboard shortcut (Ctrl + Enter)
- 📱 Responsive design
- 🌙 Dark mode support
- 🎨 Modern user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Web Speech API
- Clipboard API
